# Amazon Sales & Reviews – Data Science Project

## Beskrivning
Detta projekt analyserar Amazon-produkter med fokus på pris, rabatter och kundbetyg.
Analysen presenteras i en interaktiv Streamlit-app.

## Dataset
Datasetet kommer från Kaggle:
https://www.kaggle.com/datasets/nadaarfaoui/cleaned-amazon-sales-and-reviews-dataset

## Tekniker
- Pandas
- Matplotlib
- Seaborn
- Streamlit

## Kör appen lokalt
```bash
pip install -r requirements.txt
streamlit run app.py
